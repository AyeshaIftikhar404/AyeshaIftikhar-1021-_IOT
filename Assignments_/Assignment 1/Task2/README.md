## Pin Map
| Component | ESP32 Pin | Notes                 |
| --------- | --------- | --------------------- |
| Button    | GPIO 4    | Input, `INPUT_PULLUP` |
| LED       | GPIO 2    | Output                |
| Buzzer    | GPIO 5    | Output                |
| OLED SDA  | GPIO 21   | I2C SDA               |
| OLED SCL  | GPIO 22   | I2C SCL               |
